## الميزات الجديدة

### ١. محرّك التقارير المالية
- `src/utils/financialReports.js` — حسابات محاسبية احترافية:
  - **قائمة الدخل:** الإيرادات → COGS → إجمالي الربح → مصاريف تشغيلية → هدر → صافي الربح
  - **الميزانية العمومية:** الأصول (نقد + ذمم مدينة + مخزون)، الخصوم (ذمم دائنة)، حقوق الملكية
  - **١٠ مؤشرات أداء:** هامش إجمالي، هامش صافي، ROI، دوران المخزون، DSO، أيام الصمود، نسبة السيولة، دوران النقد، نسبة المصروفات، متوسط قيمة الطلب

### ٢. مبدّل الأوضاع الثلاثي
- ٣ أوضاع في الإعدادات: **اليومي** (بسيط) / **المدير** (احترافي) / **المستثمر** (تنفيذي)
- وضع المستثمر:
  - يخفي شريط التنقّل + FAB + POS
  - يعرض لوحة تنفيذية للقراءة فقط
  - خلفية بيضاء ناصعة `#FFFFFF` + كتل تِراكوتّا جريئة
  - انتقال ٤٠٠ms سلس
- العودة لليومي/المدير تستعيد كل الأزرار والشريط السفلي

### ٣. لوحة المستثمر (SOP §13)
- **هرم المؤشرات:** ٢ بطاقات بطل (تِراكوتّا، أرقام ٣٤px Mono) + ٤ متوسطة + ٤ صغيرة
- **قائمة الدخل الشلالية:** أشرطة نسبية بألوان دلالية (تِراكوتّا/أحمر/أخضر/ذهبي)
- **الميزانية:** ٣ بطاقات مكدّسة (أصول/خصوم/ملكية) مع أشرطة تركيبية
- **زر PDF:** يُصدّر تقريراً احترافياً (A4) يحتوي قائمة الدخل + الميزانية + ٥ مؤشرات
- **زر واتساب:** بجانب مؤشر DSO لإرسال تذكيرات تحصيل فورية

### ٤. مطابقة SOP §13
- خلفية `#FFFFFF` ناصع (ليس وضعاً داكناً)
- بطاقات البطل بتِراكوتّا `#CC785C` + نص عاجي `#FAF9F5`
- جزيرة الرقم الأهم بحبر دافئ `#2A2521` + أرقام بيضاء
- كل الأرقام IBM Plex Mono + `tabular-nums`
- تباين WCAG AA على أبيض
- احترام `prefers-reduced-motion`
